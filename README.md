# Student Gradebook System

Java-based gradebook system implementing weighted grading categories,
letter grade assignment, and GPA calculation.

## Features
- Category-weighted scoring (Homework, Quizzes, Midterm, Final)
- Accurate letter grade mapping
- GPA calculation weighted by credit hours
- Missing-category exclusion
- Clean service-driven architecture
- Comprehensive unit tests

## Build & Test
mvn clean test
